# Databases and SQL 

In this course, I developed a solid foundation in **SQL and relational databases**, starting with the basics of querying data from tables. I learned how databases are structured and how to use SQL to retrieve, filter, and manipulate data effectively. As I progressed, I gained a deeper understanding of relational database concepts, including how to create tables and modify their contents. I also explored more advanced querying techniques such as grouping, sorting, using functions, and working with string patterns, which allowed me to perform more complex data analysis and extract meaningful insights from datasets.

Additionally, I learned how to **integrate SQL with Python** to access and work with databases programmatically, particularly within environments like Jupyter Notebook. This helped me bridge the gap between data storage and data analysis. Through the course assignment, I applied my skills to real-world datasets, reinforcing my ability to solve practical problems using SQL. I also gained exposure to advanced SQL concepts such as views, stored procedures, and ACID transactions, which are important for data engineering and maintaining reliable database systems. Overall, this course equipped me with both the theoretical knowledge and practical skills needed to work confidently with databases in data-driven projects.
