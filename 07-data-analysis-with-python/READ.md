# Data Analysis with Python

## Data Understanding and Preparation

I developed foundational skills in **Python-based data analysis**, focusing on how to understand datasets and prepare them for analysis. I learned how to interpret dataset structures, identify key variables, and work with real-world datasets like laptop and used car pricing data. I also became familiar with essential Python libraries such as Pandas, NumPy, and visualization tools like Matplotlib and Seaborn, understanding their roles in a data analysis workflow. Additionally, I gained practical experience importing and exporting data in formats such as CSV, Excel, and JSON, and working within environments like Jupyter Notebook.

Key skills I developed include:

* Importing and exporting datasets using Python
* Summarizing and previewing data using Pandas
* Understanding dataset structure and variables
* Using Python’s `sqlite3` to query databases
* Working with real datasets like laptop pricing and used car pricing

## Data Wrangling

I learned how to handle missing values, correct data types, and standardize datasets to improve accuracy and consistency. I also explored techniques such as normalization, binning, and encoding categorical variables, which are essential for preparing data for analysis and machine learning. Through hands-on practice, I applied these techniques to real datasets, reinforcing the importance of structured data preparation in the data science workflow.

Some of the key techniques I applied include:

* Handling missing values (removing or replacing data)
* Converting data types for consistency
* Normalizing numerical data (min-max scaling, Z-score)
* Binning continuous variables into categories
* Encoding categorical variables (label encoding, one-hot encoding)

## Exploratory Data Analysis (EDA)

I learned how to **analyze and understand data through statistical and visual techniques**. I explored how to calculate descriptive statistics such as mean, median, and standard deviation to understand data distribution. I also learned how to identify patterns, trends, and relationships between variables using grouping and visualization techniques. This module emphasized the importance of EDA as a critical step before building models, helping to uncover insights and detect anomalies in the data.

Key EDA concepts and skills I gained include:

* Using summary statistics to understand data distribution
* Applying `groupby()` in Pandas for comparisons
* Visualizing data using plots and charts
* Analyzing relationships using correlation (e.g., Pearson correlation)
* Testing relationships between categorical variables using chi-square tests

Overall, these modules strengthened my ability to **prepare, clean, and analyze data using Python**, giving me a solid foundation for more advanced data science tasks such as modeling and machine learning.

## Model Development and Regression Analysis

In this module, I explored the fundamentals of **predictive model development using Python**, focusing mainly on regression techniques used in data analysis and machine learning. I learned how to build different types of regression models, including simple linear regression, multiple linear regression, and polynomial regression, using scikit-learn. I also gained an understanding of the machine learning workflow, including how to distinguish between independent (input) and dependent (target) variables when building predictive models. Through practical examples using datasets like used car and laptop pricing, I developed skills in applying models to real-world problems.

Key concepts and techniques I learned include:

* Building regression models (linear, multiple, and polynomial regression)
* Using pipelines to streamline model building and preprocessing
* Interpreting model outputs using residual plots and performance visualizations
* Evaluating model accuracy using metrics like R-squared and Mean Squared Error (MSE)
* Using kernel density estimation (KDE) plots to compare predicted vs actual values
* Applying regression models for real-world prediction tasks (e.g., pricing estimation)


## Model Evaluation and Decision Making

Beyond building models, I learned how to **evaluate and interpret model performance to support data-driven decision-making**. I explored how to assess how well a model fits the data using visual tools such as residual plots and distribution comparisons. I also learned how to interpret performance metrics like R² and MSE to determine model accuracy and reliability. This helped me understand that model development is not just about training algorithms, but also about ensuring they produce meaningful and reliable results.

Through hands-on labs, I applied these skills to:

* Predict used car prices using regression models
* Estimate laptop prices based on feature variables
* Compare model predictions against actual values to assess accuracy
* Improve model performance using better feature selection and transformations

## Model Optimization and Generalization

I developed advanced skills in **improving and optimizing predictive models** to ensure they perform well on new, unseen data. I learned how to identify and address common modeling issues such as **overfitting and underfitting**, and how these relate to the bias-variance trade-off. I also explored how model performance can differ between training and testing datasets, and why this difference is important for evaluating generalization ability. A key focus was understanding how to refine models so they are not only accurate on training data but also reliable in real-world applications.

I also learned important techniques for model improvement, including:

* Using **ridge regression** to reduce overfitting through regularization
* Applying **grid search** to tune hyperparameters and find optimal model settings
* Comparing model performance using training and test scores
* Evaluating models using metrics such as R² and Mean Squared Error (MSE)
* Optimizing regression models for datasets like used car and laptop pricing
