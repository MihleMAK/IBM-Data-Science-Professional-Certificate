## Tools for Data Science

In this set of modules, I learned about the **essential tools and technologies used in data science** and how they fit into a data scientist’s workflow. I explored different categories of tools, including open-source, cloud-based, and commercial options, and understood how to choose the right tools depending on the task. I also gained insight into key programming languages used in data science such as Python, R, SQL, and others, along with the criteria for selecting a language based on project needs and user roles. This helped me understand that data science is not limited to one language or tool, but rather involves a flexible toolkit tailored to solving specific problems.

Additionally, I developed a deeper understanding of **practical data science components such as libraries, APIs, datasets, and development environments**. I learned how libraries support tasks like data analysis, visualization, and machine learning, and how APIs (such as REST APIs) enable communication between systems. I became familiar with working in interactive environments like Jupyter Notebook and JupyterLab, as well as using RStudio for statistical computing and visualization. Furthermore, I gained practical knowledge of version control using Git and collaboration through GitHub, including managing repositories, branches, and merges. Overall, these modules equipped me with both the technical tools and collaborative practices needed to work effectively in data science projects.
