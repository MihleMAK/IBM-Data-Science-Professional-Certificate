# Machine Learning with Python

This course introduced me to the **core concepts, techniques, and workflows of machine learning**, focusing on how models are built, trained, evaluated, and improved using Python. I learned that machine learning is an **iterative process**, where models are continuously refined through different stages of development. The course also gave me insight into the role of a machine-learning engineer and introduced key tools such as scikit-learn, which is widely used for building and testing models in Python.

---

# Linear and Logistic Regression**

This module introduced me to two fundamental statistical learning methods used in machine learning.

I learned that:

* **Linear regression** is used to model continuous relationships between variables
* **Logistic regression** is used for classification problems
* Both models are foundational but have limitations in handling complex real-world data
* Modern ML methods often extend or improve upon these basic approaches

This helped me understand the historical foundation of predictive modeling and why more advanced algorithms are needed.

---

## Supervised Learning Models

In this module, I explored more advanced supervised learning techniques used for prediction and classification tasks.

**Key concepts learned:**

* Binary and multiclass classification
* Decision trees and regression trees
* K-Nearest Neighbors (KNN) and Support Vector Machines (SVM)
* Bias-variance trade-off and model complexity
* Strategies to reduce overfitting and improve generalization

This module helped me understand how different algorithms make predictions and how model performance can be optimized.

---

## Unsupervised Learning Models

This module focused on **learning patterns from unlabeled data**, where no target variable is provided.

I learned about:

* Clustering techniques such as k-means and hierarchical clustering
* Advanced clustering methods like DBSCAN and HDBSCAN
* Dimensionality reduction techniques such as PCA, t-SNE, and UMAP
* Feature engineering and data simplification for better modeling

These techniques helped me understand how to discover hidden patterns and structure in data without supervision.

---

## Model Evaluation and Validation

In this module, I learned how to **assess and improve machine learning models** to ensure they perform well on unseen data.

**Key skills included:**

* Evaluating models using classification and regression metrics
* Using cross-validation to test model reliability
* Hyperparameter tuning for model optimization
* Regularization techniques to reduce overfitting
* Improving model performance through systematic testing

This module emphasized the importance of building models that are not only accurate but also generalizable.

---

## Final Project and Exam

In the final module, I applied everything I learned in a **hands-on machine learning project focused on Rain Prediction in Australia**. This involved using real-world data to build, evaluate, and refine predictive models.

**Final activities included:**

* Completing a full machine learning workflow
* Applying classification techniques to real datasets
* Performing model evaluation and optimization
