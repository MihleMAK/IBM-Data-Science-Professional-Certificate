# Python for Data Science & AI 

In this course, I built a strong foundation in **Python programming for data science**, starting with the basics such as data types, variables, expressions, and string operations. I learned how to work with core data structures including lists, tuples, dictionaries, and sets, which are essential for storing and organizing data efficiently. As I progressed, I developed an understanding of key programming concepts like conditions, loops, and functions, which allow for logical decision-making and code reusability. I also explored more advanced topics such as exception handling for managing errors and object-oriented programming using classes and objects, helping me write more structured and maintainable code.

Additionally, I gained practical skills in **working with data and collecting it from different sources**. I learned how to read and write files in Python and used powerful libraries like Pandas for data manipulation and NumPy for numerical operations. I also explored how to gather data using APIs, including understanding how REST API works, and practiced basic web scraping techniques to extract data from websites. Overall, this course equipped me with both the programming fundamentals and practical data-handling skills needed to start solving real-world data problems using Python.
