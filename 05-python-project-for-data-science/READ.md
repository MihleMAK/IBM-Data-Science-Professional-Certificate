# Project 1: Extracting and analyzing stock market data using Python

This project focuses on **extracting and analyzing stock market data using Python** to support data-driven decision-making in a financial context. I learned that stocks represent ownership in a company, and their prices change based on factors like company performance, market demand, and future profit expectations. Working from the perspective of a data scientist in a hedge fund, the aim is to **identify patterns and detect potentially suspicious stock activity** using real financial data. This helped me understand how stock prices are tracked continuously and how investors make profits or losses based on market movements.

To carry out the project, I used the yfinance library together with Pandas to extract and structure stock data. Key tasks in the project included:

- Extracting **company (stock) information**
- Retrieving **historical share price data**
- Collecting **historical dividend data**
- Converting the data into **dataframes for analysis**

Overall, this project strengthened my ability to work with real-world financial datasets and showed how Python can be used in finance for data extraction, analysis, and decision-making.

# Project 2: Extracting and visualizing stock data

This project focuses on **extracting and visualizing stock data** to support better decision-making through data analysis. I learned that simply collecting data is not enough—being able to **present it clearly using visualizations** is essential in data science. The project builds on financial data concepts by working with stock prices and company revenue, helping me understand how trends and patterns can be identified through graphs. It also reinforced the importance of combining multiple data sources to gain deeper insights into a company’s performance.

To complete the project, I used tools like yfinance, Pandas, and web scraping techniques to extract and analyze data. Key tasks included:

- Defining a function to **create and display graphs**
- Using **yfinance to extract stock price data**
- Applying **web scraping to collect revenue data** for companies like Tesla and GameStop
- Cleaning and organizing the data for analysis
- Plotting **stock price vs revenue graphs** to visualize trends

Overall, this project improved my ability to work with real-world datasets, combine different data collection methods, and create meaningful visualizations that communicate insights effectively.
