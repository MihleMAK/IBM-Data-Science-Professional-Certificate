# Data Visualization with Python

## Introduction to Data Visualization

I learned the fundamentals of data visualization and why it is important for analyzing large datasets. I explored the history and structure of Matplotlib and practiced creating basic line plots using datasets such as Canadian immigration data. I also learned how to analyze data using Pandas before visualizing it.

**Key skills learned:**

* Understanding the importance of data visualization
* Creating basic plots using Matplotlib
* Working with datasets in Jupyter Notebook
* Performing simple exploratory analysis before visualization

---

## Basic and Specialized Visualizations

I learned how to represent data using different chart types depending on the type of insight required.

**Key visualizations covered:**

* Area plots
* Histograms
* Bar charts
* Pie charts
* Box plots
* Scatter and bubble plots

I also learned how each visualization type helps highlight different aspects of data, such as distribution, comparison, and relationships.

---

## Advanced and Geospatial Visualizations

I explored more advanced visualization techniques and geospatial analysis. I learned how to create more expressive visuals like word clouds and waffle charts, and how to use Seaborn for regression analysis.

**Key skills included:**

* Creating regression plots using Seaborn
* Building word clouds and waffle charts
* Visualizing geospatial data using Folium
* Creating maps with markers and choropleth layers

This helped me understand how location-based and advanced visualizations can reveal deeper insights in data.

---

## Interactive Dashboards

I learned how to use Plotly for creating interactive charts and Dash for building full web-based dashboards.

**Key concepts:**

* Creating interactive charts (scatter, bar, line, pie, etc.) using Plotly
* Building dashboard layouts using Dash components
* Adding interactivity using callbacks and user inputs
* Designing dashboards that communicate insights effectively

---

## Final Project and Assessment

I applied all my skills to a **real-world data analysis project on automobile sales during recession and non-recession periods**. I created multiple visualizations and built an interactive dashboard to present my findings. This project tested my ability to combine data analysis, visualization, and storytelling into a complete solution.

**Final project tasks included:**

* Cleaning and analyzing historical automobile sales data
* Creating multiple visualization types
* Building an interactive dashboard
* Presenting insights through visual storytelling
* Completing a final graded assessment
